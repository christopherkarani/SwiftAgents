# SwiftAgents Build Errors Summary

## Build Command
```bash
swift build
```

## Error Categories

### 1. Logger Privacy Parameter Errors (Multiple Files)
**Files Affected**:
- SwiftDataMemory.swift (lines 79, 95, 110)
- VectorMemory.swift
- ConversationMemory.swift
- Other memory implementations

**Error**: `extra argument 'privacy' in call` and `cannot infer contextual base in reference to member 'private'`

**Pattern**:
```swift
Log.memory.error("Message: \(error, privacy: .private)")
```

**Root Cause**: The Swift OSLog Logger API doesn't support `privacy:` parameter in string interpolation in the current configuration.

### 2. ObjectIdentifier Type Constraint Errors
**File**: ResilientAgent.swift
**Lines**: 130, 350

**Error**: `argument type 'X' expected to be an instance of a class or class-constrained type`

**Problematic Code**:
```swift
ObjectIdentifier(base)  // base is 'any Agent' (protocol type)
ObjectIdentifier(self)  // self is 'Self' (protocol type)
```

**Root Cause**: ObjectIdentifier requires class instances, but Agent protocol types aren't classes.

### 3. Actor Initializer Error
**File**: ResilientAgent.swift
**Line**: 83

**Error**: `'nonisolated' on an actor's synchronous initializer is invalid`

**Problematic Code**:
```swift
nonisolated init(wrapping resilient: ResilientAgent, ...)
```

**Root Cause**: Actor initializers in Swift 6 cannot be marked as nonisolated.

### 4. Duplicate Extension Error
**File**: ResilientAgent.swift
**Line**: 534

**Error**: `invalid redeclaration of 'timeInterval'`

**Problematic Code**:
```swift
extension Duration {
    var timeInterval: TimeInterval { ... }
}
```

**Root Cause**: Duration.timeInterval already exists in Foundation.
